Supprimer votre compte dans l'application MKParis Concept

Voici comment supprimer votre compte dans notre application mobile :

  1. Ouvrez l'application MKParis Concept sur votre appareil.
  2. Accédez à votre profil ou à vos paramètres.
  3. Recherchez l'option "Supprimer mon compte" ou quelque chose de similaire.
  4. Cliquez sur cette option pour commencer le processus de suppression de compte.
  5. Confirmez la suppression de votre compte lorsque vous y êtes invité.
